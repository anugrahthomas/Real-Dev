Stream: sequence of data in chunks, not all at once.

Types of Streams:
1. Readable Stream
2. Writable Stream
3. Dupex Stream: Can read and write (tcp/socket)
4. Transform Stream: special duplex stream that modifies data while passing it through
  - Compression
  - Encryption
  - Uppercasing text
eg:
const { Transform } = require("stream");
const upperCase = new Transform({
  transform(chunk, encoding, callback) {
    this.push(chunk.toString().toUpperCase());
    callback();
  }
});

üö¶ Backpressure (Very Important)
Backpressure happens when:
- Writable stream cannot handle data as fast as readable stream produces it.
Node handles this using:
- readable.pipe(writable)

.pipe() automatically:
- Manages flow
- Prevents memory overload
- Handles backpressure
